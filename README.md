# Wuxia Web3D Demo

Three.js WebGPU demo with a Blender FBX-to-GLB asset pipeline.

## Commands

```powershell
npm install
npm run assets:glb
npm run dev
npm run build
```

The demo requires a browser with WebGPU enabled. GitHub Pages serves over HTTPS, which satisfies WebGPU's secure-context requirement.

Controls: `Space` moves forward, `ArrowLeft` and `ArrowRight` rotate.
